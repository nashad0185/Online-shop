# AuraBoutique

*AuraBoutique* is a modern and elegant online shop dedicated to women's fashion and accessories.  
This project was developed as a personal final project to demonstrate proficiency in *HTML, **CSS, and **JavaScript* — showcasing front-end development   interactive web functionality.

---

## Overview

AuraBoutique is a fully responsive and visually appealing online store designed for showcasing and selling women’s clothing and fashion items.  
The website provides a seamless browsing experience with multiple pages, each serving a unique purpose — from exploring products to managing the shopping cart..

---

## Pages Structure

The website contains *6 main pages*:

1. 🏠 *Home* — A welcoming landing page introducing AuraBoutique’s style and highlights.  
2. 👗 *About* — Information about the brand and its mission to empower women through fashion.  
3. 🛍 *Products* — A product showcase featuring various women’s clothing and accessories.  
4. 💬 *Testimonials* — Real or sample customer feedback to build trust and engagement.  
5. ✉ *Contact* — A contact form for inquiries and customer support.  
6. 🛒 *Cart* — A functional shopping cart where selected items are displayed and managed.

## Features

- Fully responsive design for desktop and mobile.  
- Interactive navigation bar linking all pages.  
- Product listing with image, name, and price.  
- Add-to-cart and cart management functionality.  
- Smooth scrolling and modern animations.  
- Simple and clean user interface with elegant color palette.
